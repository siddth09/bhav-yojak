# Bhav Yojak (भाव योजक)  
### *Connecting Emotion Through Language*

**Bhav Yojak** is a real-time voice translation web application that bridges the gap between local Indian dialects—such as Bhojpuri, Awadhi, Maithili, and Haryanvi—and English/Hindi.  
It leverages the browser’s native **Speech Recognition**, combined with powerful AI-based translation and native **Text-to-Speech (TTS)** narration.

---

## ✨ Features

- 🎙️ **Continuous Speech Recognition**  
  Speak naturally without interruptions or timeout constraints.

- 🎯 **“T-Mark” Translation**  
  Explicitly trigger paragraph-by-paragraph translation using the **T-Mark** button for better contextual accuracy.

- 🔊 **AI Narration**  
  High-quality, native-script voice output using AI-powered Text-to-Speech (TTS).

- 🔐 **Bring-Your-Own-Key (BYOK)**  
  Users enter their own API key directly in the UI.  
  API keys are stored locally in the browser and are never transmitted to any external server.
